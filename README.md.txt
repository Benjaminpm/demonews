¿Qué hace?
Inicio de sesión, registro, verificación de correo electrónico, autenticación como usuario regular, como blogger y como administrador. 

¿Qué es?
Demo con la tecnología Laravel 6.2.0 y proporciona el punto de partida perfecto para su próxima aplicación de Laravel.

¿Qué tecnologías se utilizan?
Laravel Framework 6.2.0 - Carbon - Nexmo.

¿Cuál es la ambición del proyecto?
Probar las tecnologías anteriormente mencionadas y practicar Laravel Framework 6.2.0. 

¿Se implementará en algún lugar?
No. Sin embargo proporciona el punto de partida perfecto para su próxima aplicación de Laravel.

¿Cuál es la etapa del proyecto?
Proyecto en curso, sin embargo no se tiene actualizaciones por el momento, ya que se trabaja en paralelo con otro proyecto personal.

¿Hay algunos problemas conocidos o cosas que no se hacen correctamente?
Sí. Se debe implementar Laravel Framework 5.8.0 en adelante, ya que con versiones anteriores surgen inconvenientes al momento de crear la autenticación.
A tener en cuenta:
Para publicar una noticia debe haber iniciado sesión.
Cuando inicia sesión tiene acceso a editar y eliminar noticias de todos los usuarios, es el punto donde va el proyecto, aquí lo que sigue es filtrar las noticias por el ID de usuario logueado, con esto se tendrá acceso a editar y eliminar únicamente a las noticias creadas por dicho usuario.
